# Mineração de Dados 🤖 Engineering & Data Science

Este repositório centraliza meus projetos e estudos práticos desenvolvidos na disciplina de **Mineração de Dados** (9º período) no **IFPB - Campus Campina Grande**. O foco aqui não é apenas o domínio dos dados, mas a aplicação de rigor metodológico, engenharia de software e otimização de modelos.

## 🛠️ Stack Tecnológica

- **Core:** Python, Pandas, NumPy
- **Machine Learning:** Scikit-Learn (Modelos Lineares, Ensembles, SVM)
- **Engenharia de Modelos:** Pipeline de Processamento, GridSearchCV, Validação Cruzada
- **Visualização:** Seaborn, Matplotlib (EDA Multivariada)

---

## 📂 Portfólio de Aplicações

### 🧬 Saúde: Classificação Oncológica

Aplicação de modelos preditivos para diagnóstico de tumores de mama.

- **Destaque Técnico:** Tuning de hiperparâmetros para maximização de **Recall**, garantindo segurança no diagnóstico médico.
- **Modelos:** Regressão Logística com Regularização (ElasticNet), SVM e Random Forest.

### ⚽ Esporte: Análise de Torneios Internacionais

Processamento e análise de grandes volumes de dados históricos de competições globais.

- **Destaque Técnico:** Engenharia de Atributos (_Feature Engineering_) e tratamento de variáveis categóricas (Label Encoding/One-Hot) em datasets de torneios (FIFA World Cup, UEFA Euro, etc.).
- **Objetivo:** Identificar padrões de desempenho e tendências competitivas ao longo das décadas.

---

## 📐 Metodologia Aplicada

Em todos os projetos, sigo um pipeline padronizado de engenharia de dados:

1. **Exploratory Data Analysis (EDA):** Identificação de padrões, correlações e outliers através de matrizes de dispersão e histogramas.
2. **Data Cleaning & Prep:** Padronização de escalas (`StandardScaler`) e codificação de variáveis.
3. **Model Selection:** Comparação sistemática entre diferentes algoritmos.
4. **Performance Tuning:** Otimização fina via busca em grade para extrair o máximo de performance métrica.
